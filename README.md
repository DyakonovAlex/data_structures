## Data Structures

Реализация базовых структур данных на Rust.

### Модули

- **`arrays`**
  - **Трейт**: `Array<T>`
  - **Реализации**:

| Реализация | Инвариант | Особенности |
|---|---|---|
| `UnsortedArray<T>` | порядок не гарантируется | быстрое удаление: на место удаляемого ставится последний элемент |
| `SortedArray<T: Ord>` | массив всегда отсортирован | быстрый `find` (бинарный поиск), но вставка/удаление требуют сдвигов |

### Интерфейс `Array<T>`

Трейт задаёт абстрактный интерфейс для массивов.

- **`insert(&mut self, item: T) -> Result<(), ArrayError>`**: вставка элемента (ошибка при переполнении).
- **`delete(&mut self, index: usize) -> Result<T, ArrayError>`**: удаление по индексу с возвратом удалённого значения.
- **`find(&self, item: &T) -> Option<usize>`**: поиск индекса по значению.
- **`get(&self, index: usize) -> Option<&T>`**: доступ к элементу по индексу.
- **`size(&self) -> usize`**: текущее количество элементов.
- **`capacity(&self) -> usize`**: максимальная ёмкость.
- **`is_empty(&self) -> bool`**, **`is_full(&self) -> bool`**: проверки состояния.
- **`traverse(&self, f: impl FnMut(&T) -> Result<(), ArrayError>) -> Result<(), ArrayError>`**: обход всех элементов с пользовательской функцией.

Ошибки описываются через `ArrayError`:

- `Full`, `Empty`, `IndexOutOfBounds`.

### Асимптотика (Big O)

Ниже — асимптотика по времени и по **дополнительной** памяти (не считая хранения самих элементов).

| Операция | `UnsortedArray` время | `SortedArray` время | Доп. память |
|---|---:|---:|---:|
| `insert` | \(O(1)\) | \(O(n)\) | \(O(1)\) |
| `delete(index)` | \(O(1)\) (swap с последним) | \(O(n)\) (сдвиг) | \(O(1)\) |
| `find(item)` | \(O(n)\) | \(O(\log n)\) | \(O(1)\) |
| `get(index)` | \(O(1)\) | \(O(1)\) | \(O(1)\) |
| `size/capacity/is_empty/is_full` | \(O(1)\) | \(O(1)\) | \(O(1)\) |
| `traverse` | \(O(n)\) | \(O(n)\) | \(O(1)\) |

Память структуры для обеих реализаций: **\(O(n)\)**, где \(n = capacity\) (фиксированный буфер `Box<[Option<T>]>` + поле `size`).

### Примеры

В проекте есть примеры:

```bash

```

Оба примера показывают базовые операции (`insert`, `delete`, `find`, `get`) и использование `traverse`.
